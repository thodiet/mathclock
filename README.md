  This app allows you to configure a widget that displays a digital clock with the time written in German words below it.
  The word clock is based on quarter-hour intervals, a style particularly common in Southern Germany.
  In addition to the quarter-hour basis, a 5-minute basis is also available.
  The word clock’s style and font can be customized.
  The widget’s transparency is fully adjustable.
  Additionally, both the digital clock and the word clock displays can be toggled off individually.
